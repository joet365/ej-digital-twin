# Kate's AI Receptionist Backend  API for the Conquer365 voice AI platform.

## Deployment

This project is deployed on Vercel.

## Environment Variables

Required environment variables:
- `SUPABASE_URL`
- `SUPABASE_SERVICE_KEY`
- `OPENAI_API_KEY`
- `FIRECRAWL_API_KEY`
- `VAPI_API_KEY`
- `PYTHON_SCRIPT_PATH`

See `.env.example` for details.
